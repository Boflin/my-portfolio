HTML

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

HTML describes the structure of a web page.

HTML elements are represented by tags.

For example, <p> is a tag for a paragraph.

CSS

CSS stands for Cascading Style Sheets.

CSS controls the layout of multiple web pages all at once.

CSS can be added to HTML documents in three ways:

  • Inline styles
  • Internal styles
  • External styles

With CSS, you can change the color of text, style lists, and add background images.

For example, to change text color, you can use the color property.

JavaScript

JavaScript is a programming language that allows you to implement complex features on web pages.

It enables interactive web pages and is an essential part of web applications.

JavaScript can manipulate HTML and CSS to update content, validate forms, and much more.

For example, to create a function in JavaScript, you can use the function keyword.

APIs

An API (Application Programming Interface) allows different software applications to communicate with each other.

APIs can be used to retrieve data from a server or send data to a server.

REST and GraphQL are common types of APIs.

For example, you can fetch data from an API using JavaScript's fetch() method.

Version Control

Version control systems are tools that help you manage changes to source code over time.

They allow you to track changes, revert to previous states, and collaborate with others.

Git is a popular version control system.

To initialize a Git repository, you can use the command git init.
